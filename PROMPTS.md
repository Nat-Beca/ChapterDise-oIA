# 🤖 Prompts para Claude — Genera tu arte SVG

Copia cualquiera de estos prompts en [claude.ai](https://claude.ai), personaliza las partes entre `[ ]` y pega el SVG resultante en el sandbox de tu tarjeta.

> **Regla de oro:** pídele solo el código SVG, sin explicaciones, y que funcione inline en HTML.

---

## 🌌 Espacio y geometría

```
Genera un SVG animado de 120x120px para una tarjeta de perfil de diseñador.
Estilo: geometría espacial con planetas y órbitas.
Mi color favorito es [AZUL ELÉCTRICO / MORADO / etc.].
Úsalo como color principal con variaciones más oscuras y claras.
Solo devuelve el código SVG sin explicaciones, que funcione inline en HTML.
Sin texto, solo formas geométricas animadas.
```

---


## 🛠 Tips para mejores resultados

| Problema | Solución |
|----------|----------|
| El SVG es muy grande o muy chico | Agrega: "el SVG debe caber en 120x120px sin recortes" |
| Tiene texto con tu nombre | Agrega: "sin texto ni labels dentro del SVG" |
| La animación es muy rápida | Agrega: "animaciones lentas, entre 3 y 6 segundos por ciclo" |
| No se ve bien sobre mi fondo | Agrega: "fondo transparente, sin `<rect>` de relleno completo" |
| Quiero otro intento | Escríbele: "hazlo de nuevo pero más [simple / colorido / detallado]" |
